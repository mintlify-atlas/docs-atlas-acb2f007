## Overview

The Done hub captures projects that have reached a natural conclusion. Work lands here when an initiative is fully delivered, a goal is met, or a project is intentionally closed. Preserving completed work in a dedicated hub keeps the active project hubs focused while ensuring outputs remain accessible.
