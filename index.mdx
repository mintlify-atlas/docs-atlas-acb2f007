## How this is organized

Each category contains one or more project hubs. A hub groups all Genspark chats related to a single project or topic. Inside each hub you'll find:

- A summary of what was researched and why
- Key outputs from each chat session
- Links to relevant Notion documents or Wiki.js pages where applicable

<Note>
  Some hubs are empty — they represent planned research areas not yet explored. Active hubs are marked with chat counts on their overview pages.
</Note>
