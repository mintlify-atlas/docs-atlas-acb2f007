## Quick reference

<AccordionGroup>
  <Accordion title="What is a Genspark hub?">
    A Genspark hub is a curated collection of AI chat sessions organized around a topic. This knowledge base extracts and structures the most useful outputs from those sessions into navigable documentation pages.
  </Accordion>
  <Accordion title="How do I add new chats?">
    Run a new chat session in Genspark, export or copy the output, and add it to the relevant hub page in this knowledge base. Follow the existing page structure to keep content consistent.
  </Accordion>
